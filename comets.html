The Comets

A 3D Data Visualization

The Comets is an on going data visualization project that I am working on. I wanted to explore 3D data visualizations and an astronomical data set was a great place to start. Nasa has collected orbital data on Near-Earth Comets . I began by exploring the data and prototyping the layout of the project using d3.js. I will be updating my progress on the project in the coming weeks!
